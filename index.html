Prework Study Guide

HTML

  • HTML is used to add structure to our web app.
  • Page content we would like the viewer to see will go in the main element.
  • Title, meta, and alt elements are useful for SEO.
  • The internet is "cool"

CSS

  • CSS is used to add style to our web app.
  • It stands for Cascading Style Sheets.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use a class selector instead of an element name for greater control as to which elements are affected by a style attribute.

Git

  • Git allows us to implement version control when developing a web app.
  • GitHub allows multiple collaborators to contribute to a web app, remotely.

JavaScript

  • JavaScript allows us to add logic and user interactivity to our web app.
  • JavaScript and Java, an OOP language, are unrelated.
